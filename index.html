O mne

Som programátor samouk so záujmom o technológie a vývoj softvéru. Ťahá ma to skôr k backendu, z programovacích jazykov najlepšie ovládam Python, HTML, truchu CSS a JavaScript. Na základnej úrovni Java / Kotlin. Dobre si rozumiem s frameworkom Django a mám praktické skúsenosti s MySQL, GIT / GitHub a Flask. Rád a rýchlo sa učím nové technológie, z predchádajúcich pozícii som si priniesol zládanie práce pod tlakom, odoslnoť voči stresu a schopnosť riešiť problémy všetkého druhu.
