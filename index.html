O mne

Som človek so záujmom o technológie a vývoj softvéru. Inklinujem k backendu, z programovacích jazykov ovládam dobre Python a HTML, čiastočne CSS a JavaScript. Na základnej úrovni Java / Kotlin. Rozumiem si tiež s frameworkom Django a mám praktické skúsenosti s MySQL, GIT / GitHub, Flask a rôznymi API. Rád a rýchlo sa učím nové technológie, z predchádajúcich pozícii si prinášam zvládanie práce pod tlakom, odoslnoť voči stresu a schopnosť riešiť problémy všetkého druhu. Na Codewars mám viac ako 250 vyriešených problémov v Pythone a na W3Schools.com som doteraz nazbieral cez 31.000 bodov za rôzne kurzy ktoré som tam absolvoval a toto číslo stále rastie nakoľko túto platformu naďalej využívam na zlepšovanie svojich schopností. Moje projekty

Cykloklub

Zatiaľ môj najväčší projekt, zameraný na celkový manažment cyklo-turisticého klubu ktorého som členom a pokladníkom. Vytvoril som ho v Django, je v ňom implementované prihlasovanie/registrácia, emailová notifikácia, grafy, galéria cez databázu, prihlasovanie členov na akcie a pár ďalších vecí. Pre viac info si pozrite README.md na GitHube. Slabšou stránkou projektu je zatiaľ frontend a aj preto sa momentálne snažím zlepšit v CSS / JavaScript.

Open-Meteo do MySQL

Flasková aplikácia ktorá sťahuje meteorologické dáta zo stránky Open-Meteo pomocou ich API, ukladá tieto údaje do MySQL databázy a následne ich zobrazí v tabulke na stránke. Zjednodušená verzia (čistý Python kód je v repozitári na adrese https://github.com/emax-77/OpenMeteo-to-MySQL-PythonOnlyVersion)
